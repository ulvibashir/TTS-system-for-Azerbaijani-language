# Master's Dissertation Project

**Institution:** Azerbaijan State Economic University (UNEC)
**Program:** MBA in Artificial Intelligence (2nd Course)
**Topic:** Design and development of a rule-based text-to-speech (TTS) system for Azerbaijani language
**Target:** 51-75 pages
**Citation Style:** APA 7

---

## Project Overview

This repository contains all materials, research, implementation, and writing for my master's dissertation on developing a rule-based TTS system for the Azerbaijani language.

### Research Focus
- Rule-based text-to-speech architecture
- Azerbaijani phonetics and phonology
- Grapheme-to-phoneme (G2P) conversion rules
- Prosody and stress assignment
- System evaluation and results

## Workflow & Best Practices

### Daily Routine
1. Check **00_Planning/DEADLINES.md** for upcoming milestones
2. Update **00_Planning/NOTES.md** with daily progress
3. Log any new papers in **00_Planning/REFERENCES.md**
4. Track chapter progress in **00_Planning/OUTLINE.md**

### When Starting Research
1. Save PDFs to `01_Research/PDFs/` with clear names (e.g., `RuleBasedTTS_Smith2020.pdf`)
2. Add citation to **00_Planning/REFERENCES.md**
3. Write summary in `01_Research/Notes/`

### When Writing Dissertation
1. Draft chapters in `03_Dissertation/Chapters/`
2. Save figures to `03_Dissertation/Figures/`
3. Update checkboxes in **00_Planning/OUTLINE.md**

### When Coding
1. Keep code in `02_Technical/Code/`
2. Document linguistic rules in `02_Technical/Rules/`
3. Save test results in `02_Technical/Experiments/`
